⚠ Can't Load Department Files

This page is open as . Browsers block fetch() between local files opened this way, so the department JSON files cannot be read — that's why every department shows "No submissions yet" even if the files have data.

To fix it, do one of the following:

  • For local testing: open a terminal in this folder and run python -m http.server 8000, then visit http://localhost:8000/admin-dashboard.html instead of double-clicking the file.
  • For real use: push this v2 folder to your GitHub Pages repo (same place V1 is deployed) and open it from https://varmais-lab.github.io/reports/v2/admin-dashboard.html — this works exactly like V1 does, no server needed.
