How Data Flows

  1. Each department head opens Submit Report, selects their department, and enters their PIN — verified securely against Supabase, never stored in this page's code.
  2. Weekly numbers are saved directly to a shared Supabase database — no files to create, export, or upload.
  3. The Admin Dashboard reads the same database live — the moment a department head saves, it's visible on the dashboard. No manual upload step, no GitHub commits required.
  4. Admins can also change any department's Maintenance % or reset its PIN directly from the dashboard, with changes taking effect immediately.
