How Data Flows

  1. Each department head opens Submit Report, unlocks their department with the shared PIN, and connects (or creates) their own <department>.json file.
  2. Weekly numbers are saved directly into that file on their computer.
  3. To make the numbers visible on the Admin Dashboard, the department head (or you) uploads the updated JSON file into the root of this site's repository (same folder as admin-dashboard.html) — replacing the previous version.
  4. The Admin Dashboard automatically fetches all 5 files from the repo root on load and shows everything in one page.
